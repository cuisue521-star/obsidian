## 一、基础定义

股权穿透（穿透式核查）：**不局限于工商、中登表面登记的直接股东，逐层向上拆解多层公司、合伙企业、资管、代持架构，追溯至最终实际控制人、受益所有人（UBO）**，遵循**实质重于形式**原则，看穿壳公司、嵌套、代持隐藏的真实权益归属。

简单一句话：**剥开多层持股外壳，找到背后真正说了算、拿收益的自然人 / 主体**。
## 二、股权穿透核心目的（金融风控 / 保理 / 信贷 / 证券通用）

### 1. 合规强制要求（反洗钱、监管硬性规定）

识别**受益所有人**：法规要求机构必须穿透识别持股≥25% 自然人，落实反洗钱、受益所有人备案，杜绝匿名持股、隐蔽代持。

### 2. 识别真实实际控制人，规避主体风险

- 防止企业用多层壳公司掩盖失信、被执行人、失信担保人；
- 区分名义法人 / 股东与幕后实控人，判断真实还款意愿、资产实力；
- 排查实控人是否多头负债、大额担保、涉诉。

### 3. 识别关联方，防范利益输送、关联担保圈

穿透后统一归集同一实控人旗下全部企业，识别隐性关联公司：

- 避免关联企业互相担保、虚增营收；
- 防止通过多层嵌套转移资产、掏空企业、重复融资。

### 4. 交易 / 融资尽调风控（保理、供应链金融、信贷高频）

1. 核查底层资产权属：穿透确认应收账款出让方、核心企业是否为同一控制下关联主体，防止虚假贸易；
2. 排查隐性负债：实控人旗下其他公司逾期、被执行，会传导至借款主体；
3. 判断还款兜底能力：实控人整体资产、征信、涉诉作为授信重要依据。

### 5. 证券、资管监管穿透

- 上市公司 / IPO：穿透股东，防止股东超 200 人、规避锁定期、代持入股；
- 银行理财、私募基金：向上穿透投资人、向下穿透底层资产，禁止多层通道规避监管限制v.icbc.com...。

### 6. 司法与债权保障（刺破公司面纱）

企业欠债无力偿还时，穿透至滥用有限责任的实控人，要求其承担连带清偿责任，防止老板用多家空壳公司转移资产逃债。
## 三、股权穿透 VS 中登登记 / 中登查询（三者清晰区分）

### 1. 中登登记

动作：**写入权属记录**；对象：应收账款 / 股权质押、转让确权；作用：固化权利、产生对抗效力。

### 2. 中登查询

动作：**读取登记台账**；对象：已登记资产瑕疵核查；作用：单笔资产尽调，查是否质押、转让。

### 3. 股权穿透

动作：**向上追溯主体架构**；对象：企业股东、实控人、关联企业；作用：查清**人**的背景、关联风险、整体负债，是**主体层面风控**。

### 实务联动流程（供应链保理标准流程）

1. 先做**股权穿透**：查清买方、卖方实控人，排查关联风险、失信、担保圈；
2. 再做**中登查询**：核查目标应收账款有无在先转让 / 质押；
3. 确认无风险后办理**中登登记**，锁定债权优先权。

## 极简一句话总结

- 中登登记：给资产 “上户口确权”
- 中登查询：查资产户口有没有瑕疵
- 股权穿透：查企业背后 “真正老板是谁、有多少关联风险”

## 核心逻辑

间接持股 = 每一层持股比例依次相乘

张三没有直接持有 A，他是通过一层一层公司间接拿钱、享有权益，每一层股权都要打折。

### 分步计算

1. B 占 A：60% = 0.6
2. C 占 B：70% = 0.7
3. 张三占 C：100% = 1
    
    张三间接持有 A 比例 = 0.6 × 0.7 × 1 = 0.42 = **42%**

- 一条持股线：逐层相乘
- 多条持股线：每条分别相乘，结果相加
