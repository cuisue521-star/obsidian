中登 = 中国证券登记结算有限责任公司
国内唯一法定证券登记结算机构，所有股票、债券、基金、ABS、供应链金融资产、非标债权转让、电子仓单等标准化 / 类标准化资产的权属登记、过户、信息核验均由其负责
## 一、中登登记（核心：权属确权、备案、过户，是 “存证 / 确权动作”）

### 1. 定义

指资产权利人通过业务系统，向中国结算提交资产持有、转让、质押、变更信息，由中登**官方登记在册、生成法定权属记录**的全流程，是产生有效物权 / 债权公示效力的核心环节。

### 2. 核心本质

**造记录、留权属、发生效力**，属于**写入中登数据库**的变更类业务。

### 3. 常见业务场景

1. 证券交易过户：股票买卖后，中登登记变更股东持仓；
2. 债权 / 保理资产登记（供应链金融高频）：应收账款转让、再保理、质押，在中登动产融资统一登记公示系统完成登记；
3. 资产确权备案：资管产品底层资产、收益权、ABS 底层债权登记；
4. 质押 / 解押登记：应收账款质押、股权质押备案；
5. 变更登记：权利人变更、转让、注销、终止备案。
## 二、中登查询（核心：调阅、核验已存登记数据，是 “读取动作”）

### 1. 定义

主体凭合法授权、业务资质，向中登系统调取**已完成登记的历史 / 当前权属数据**，用于核验资产真实性、排查权利瑕疵、尽调核查，仅读取数据、不修改任何登记记录。

### 2. 核心本质

**查记录、验真伪、做尽调**，纯读取操作，不改动中登底层权属台账。

### 3. 常见业务场景

1. 贷前尽调：银行 / 保理查询应收账款是否已转让、质押，防止重复融资；
2. 交易前核验：受让方查询标的资产权属是否干净、有无查封冻结；
3. 司法核查：法院、律师调取中登登记档案用于诉讼；
4. 内部风控复核：企业、资管定期自查名下资产登记状态；
5. 个人 / 机构持仓查询：股东查询自身股票、基金持有明细。

## 四、中登登记的目的
- **法定确权公示**
    将应收账款、动产、股权、证券等资产转让、质押、变更信息录入官方登记系统，产生公示对抗效力，对外公示权利归属，防止一物多卖、重复融资。
- **固化权利凭证**
    形成官方可留存、可核验的电子化权属记录，作为交易、融资、诉讼的法定证据。
- **锁定资产权利状态**
    完成转让 / 质押登记后，第三方在后设立的权利不能优先于已登记权利，保障受让方、质权人优先受偿。
- **完成交易生效要件**
    保理、再保理、资产转让、质押业务中，登记是风控闭环必备环节，未登记无法对抗善意第三人。
- **规范资产流转台账**
    统一留存资产全生命周期变更记录，实现资产流转全程可追溯。
## 五、中登查询的目的
- **前置风险排查（贷前 / 交易前）**
    核查标的资产是否已有转让、质押、查封、异议登记，排查重复融资、隐性负债风险。
- **核验权属真实性**
    确认登记主体、债权金额、期限、登记状态与交易合同一致，杜绝虚假资产。
- **事后风控复核**
    放款 / 资产受让完成后，定期查询核对登记状态，监控是否出现新增权利瑕疵。
- **交易尽调取证**
    为保理、银行、信托、律所、法院提供官方登记档案，作为尽调、诉讼依据。
- **确认登记完成情况**
    办理登记后通过查询核验登记是否成功、信息录入有无错误。
## 六、中登登记和中登查询的主体
### （一）中登登记（办理登记的主体）

1. **法定主办主体：担保权人（受让方 / 保理商 / 银行 / 质权人）**
    
    法规明确：登记由*==*担保权人**发起办理==，不是融资方（卖方 / 担保人）黑龙江政务...。
    
    - 保理业务：保理公司、银行（受让应收账款一方）
    - 应收账款质押：放款银行、信托、资管
    - 融资租赁、动产抵押：出租人、抵押权人
    
<font color="#205867">1. <font color="#205867">**配合主体：担保人 / 出让人（融资企业、应收账款原债权人）**</font></font>
    
    不用操作登记，但必须与担保权人确认登记信息、配合提供资料。
2. **代办主体**
    
    担保权人可委托律所、第三方风控服务商代为操作登记。
