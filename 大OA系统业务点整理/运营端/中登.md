中登 = 中国证券登记结算有限责任公司
国内唯一法定证券登记结算机构，所有股票、债券、基金、ABS、供应链金融资产、非标债权转让、电子仓单等标准化 / 类标准化资产的权属登记、过户、信息核验均由其负责
## 一、中登登记（核心：权属确权、备案、过户，是 “存证 / 确权动作”）

### 1. 定义

指资产权利人通过业务系统，向中国结算提交资产持有、转让、质押、变更信息，由中登**官方登记在册、生成法定权属记录**的全流程，是产生有效物权 / 债权公示效力的核心环节。

### 2. 核心本质

**造记录、留权属、发生效力**，属于**写入中登数据库**的变更类业务。

### 3. 常见业务场景

1. 证券交易过户：股票买卖后，中登登记变更股东持仓；
2. 债权 / 保理资产登记（供应链金融高频）：应收账款转让、再保理、质押，在中登动产融资统一登记公示系统完成登记；
3. 资产确权备案：资管产品底层资产、收益权、ABS 底层债权登记；
4. 质押 / 解押登记：应收账款质押、股权质押备案；
5. 变更登记：权利人变更、转让、注销、终止备案。

| 对比维度   | 中登登记               | 中登查询              |
| ------ | ------------------ | ----------------- |
| 操作属性   | 写入、变更台账            | 只读、调取台账           |
| 业务目的   | 确权、转让、质押、备案，产生公示效力 | 核验资产、排查风险、尽职调查    |
| 操作顺序   | 先发生登记，后续才能查询       | 必须存在登记记录，才有查询内容   |
| 是否改变权属 | 是，直接变更资产权利状态       | 否，仅查看，不影响资产权属     |
| 使用主体   | 转让双方、融资企业、资管管理人    | 银行、保理、信托、风控、律所、法院 |
| 典型风控作用 | 固化资产归属，避免一物多卖      | 排查隐性转让、重复质押、权利瑕疵  |
