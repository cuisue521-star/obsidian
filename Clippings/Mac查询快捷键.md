### 🔍 全局搜索 (Spotlight)

`Command` + `空格键` 是 macOS 最核心的全局搜索功能，能在瞬间找到几乎所有文件、应用和系统设置[](https://support.apple.com/zh-tw/guide/mac-help/mh26783/26/mac/26)。

| 目标操作           | 快捷键                                | 说明                                                                                                                                                                        |
| -------------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 开启/关闭Spotlight | `Command` + `空格键`                  | 从任何地方呼出或关闭全局搜索框[](https://support.apple.com/zh-tw/guide/mac-help/mh26783/26/mac/26)[](https://jingyan.baidu.com/article/fec4bce2d9b0e7b3608d8b74.html)。                   |
| 查看文件完整路径       | `Command` (长按)                     | 在Spotlight结果中选中文件后长按，路径会直接显示在文件名下方[](http://support.apple.com/zh-cn/guide/mac-help/mh26783/26/mac/26)[](https://blog.csdn.net/weixin_41905135/article/details/157223680)。 |
| 打开文件所在文件夹      | `Command` + `R` 或 `Command` + `回车` | 快速跳转到文件的实际存储位置[](https://support.apple.com/zh-tw/guide/mac-help/mh26783/26/mac/26)[](https://blog.csdn.net/weixin_41905135/article/details/157223680)。                    |
### 🗂️ 访达 (Finder) 中搜索

在访达窗口中进行更精细的文件筛选和路径操作。

| 目标操作         | 快捷键                        | 说明                                                                                               |
| ------------ | -------------------------- | ------------------------------------------------------------------------------------------------ |
| 打开搜索窗口       | `Command` + `F`            | 在当前访达窗口顶部调出搜索栏，可设置筛选条件[](https://jingyan.baidu.com/article/fec4bce2d9b0e7b3608d8b74.html)。       |
| 打开“前往文件夹”窗口  | `Shift` + `Command` + `G`  | 手动输入或粘贴完整路径，实现精准跳转[](https://jingyan.baidu.com/article/fec4bce2d9b0e7b3608d8b74.html)。           |
| **显示/隐藏路径栏** | `Option` + `Command` + `P` | 在访达窗口底部**开启或关闭**路径栏，方便随时查看层级[](https://blog.csdn.net/weixin_41905135/article/details/157223680)。 |
| **查看文件简介**   | `Command` + `I`            | 弹出详细窗口，其中“位置”字段会显示完整路径[](https://blog.csdn.net/weixin_41905135/article/details/157223680)。       |
| **复制文件路径**   | `Option` + `Command` + `C` | 将所选文件的完整路径复制到剪贴板[](https://blog.csdn.net/weixin_41905135/article/details/157223680)。             |

|目标操作|快捷键|说明|

|快速跳转至桌面|`Command` + `D`|一键定位到桌面文件夹[](https://blog.csdn.net/nokiaguy/article/details/7833053)。|
|快速跳转至应用程序文件夹|`Shift` + `Command` + `A`|一键定位到“应用程序”目录[](https://blog.csdn.net/nokiaguy/article/details/7833053)。|
